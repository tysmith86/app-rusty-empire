Server Description

Rusty Empire has non-playing admins who are ready to help when and where they can. We have weekly events that take place on admin island, which iclude but are not limited to, arena battles and art competitions. Winners get a supply signal. Rusty Empire is strictly vanilla and is on a bi-weekly wipe cycle. Our staff is held to the same rules as the rest of the community. Our Teamspeak server is rustyempire.teamspeak3.com.
